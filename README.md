# PS透视变形工具

一个基于网页的图片处理工具，实现了Photoshop透视变形功能。

## 功能特点

✨ **透视变形** - 拖动4个角点进行图片透视変形
- 4个可交互的控制点
- 实时预览变形效果
- 双线性插值平滑处理
- 接近PS的变形效果

🖼️ **图片处理**
- 支持各种图片格式上传
- 自适应显示
- 导出变形后的图片为PNG格式

⚡ **用户友好**
- 按住鼠标拖动控制点
- 鼠标释放时自动渲染
- 重置按钮快速还原
- 实时反馈

## 使用方法

1. 打开工具页面
2. 点击"选择图片"上传图片
3. 拖动四个蓝色控制点来改变图片透视
4. 点击"导出"保存变形后的图片
5. 点击"重置"恢复原始状态

## 技术实现

- **纯HTML5 Canvas** - 无外部依赖
- **透视变换算法** - 4点透视矩阵计算
- **双线性插值** - 平滑的像素采样
- **高斯消元法** - 矩阵求解
